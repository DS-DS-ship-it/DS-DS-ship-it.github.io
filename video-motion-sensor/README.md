# Video Motion Sensor

A lightweight, Safari-friendly motion tracker that runs fully in the browser.

## Features
- Real-time motion detection + bounding boxes
- Draggable/resizable Picture-in-Picture (PiP) motion zoom
- Stable in-browser recording (segmented option to keep RAM stable)
- Download last clip

## Tips
- Works best on HTTPS (GitHub Pages) or `http://localhost`
- If Safari struggles, lower **Processing width**, **Detection FPS**, or **Bitrate**
